# Ellipse



## 组件使用
[两种使用方式](/product/BizCharts4/category/61/page/181#使用方式)

## attrs属性

使用方式
```js
<Ellipse attrs={{
          x: 300,
          y: 200,
          rx: 30,
          ry: 50,
          fill: '#1890FF',
          stroke: '#F04864',
          lineWidth: 4,
          radius: 8,
        }} />
```

### 通用属性
[绘图属性](./169)

### x
_<number>_
圆心坐标的 x 坐标。

### y
_<number>_
圆心坐标的 y 坐标。

### rx
_<number>_
椭圆的水平半径。

### ry
_<number>_
椭圆的垂直半径。
